About My Tech Project

Project Overview

Welcome to my tech project! This project leverages cutting-edge technologies to deliver an interactive and educational experience. Below is a brief description of the tools and technologies used:

  • Sanic: A Python web framework that is fast, lightweight, and supports asynchronous programming.
  • MongoDB: A NoSQL database used to store user data and quiz questions.
  • Redis: An in-memory data structure store, used for caching and session management.
  • BeautifulSoup: A Python library for web scraping to fetch the latest tech news and updates.
  • Asyncio: A Python library used to write concurrent code with the async/await syntax, ensuring that the application runs smoothly and efficiently, handling multiple requests without blocking.
  • HTTPX: A fully featured HTTP client for Python 3, which provides asynchronous capabilities to interact with web services and APIs.
  • Pytest: A testing framework that makes it easy to write simple and scalable test cases for the application.
  • JavaScript: The programming language used to create dynamic and interactive user interfaces.
  • CSS: A stylesheet language used to control the presentation of HTML documents, ensuring a responsive and visually appealing design.
  • HTML: The standard markup language for creating web pages.

Features

This project offers several features aimed at tech enthusiasts:

  • Newsletter: Stay updated with the latest news and trends in the tech world right from the home page. Using web scraping, the application fetches the most recent articles from Stack Overflow.
  • Tech Quiz: Test your knowledge with quizzes designed for educational and recreational purposes, focused on the tech industry. The quizzes are dynamically fetched from a third-party API and stored in MongoDB for easy access.

Why This Project?

The main goal of this project is to provide an engaging platform for learning and staying informed about technology. Whether you're a seasoned developer or a curious beginner, there's something here for you. By combining various technologies and methodologies, this project ensures a robust and user-friendly experience.
